VideoCore IV example - App Fonts Demo

An example showing the use of the VGShapesLoadAppFont function to load custom application fonts into VGShapes and display them on multiple layers.

Example contributed by Richard Metcalfe.

Raspberry Pi 3B/3B+/3A+/Zero2W version, copy the code to a new project to create a version for A/B/A+/B+/Zero/2B.
