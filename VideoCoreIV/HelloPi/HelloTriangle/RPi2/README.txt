VideoCore IV example - Hello Triangle

A rotating 3D cube in OpenGL ES with an image rendered on each face.

This example directly imports the C code from the original hello_pi collection.

Make sure you also copy the Djenne_128_128.raw, Gaudi_128_128.raw and Lucca_128_128.raw files from the Media folder to your SD card.

Raspberry Pi 2B/3B/3B+/3A+/Zero2W version
